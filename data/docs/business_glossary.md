# Business Glossary

This glossary defines common business terms used in the Data Analyst Agent.

## Regions

| Code | Full Name | Countries |
|------|-----------|-----------|
| **NA** | North America | USA, Canada |
| **EMEA** | Europe, Middle East, Africa | UK, Germany, France, etc. |
| **APAC** | Asia-Pacific | Japan, Australia, Singapore, etc. |
| **LATAM** | Latin America | Brazil, Mexico, Argentina, etc. |

---

## Customer Status

| Status | Description |
|--------|-------------|
| **Active** | Currently paying subscriber with valid subscription |
| **Churned** | Cancelled subscription, no longer paying |
| **Trial** | In free trial period, not yet converted |
| **Suspended** | Temporarily disabled (usually payment issues) |

---

## Subscription Plans

| Plan | Description | Typical MRR |
|------|-------------|-------------|
| **Starter** | Entry-level plan for small teams | $99-299/month |
| **Professional** | Mid-tier plan with advanced features | $299-799/month |
| **Enterprise** | High-tier plan with custom pricing | $999+/month |

## Time Periods

| Abbreviation | Meaning |
|--------------|---------|
| **MTD** | Month to Date (from start of current month) |
| **QTD** | Quarter to Date (from start of current quarter) |
| **YTD** | Year to Date (from start of current year) |
| **MoM** | Month over Month comparison |
| **QoQ** | Quarter over Quarter comparison |
| **YoY** | Year over Year comparison |
| **LTM** | Last Twelve Months (trailing 12 months) |
